Установка и запуск проекта:

Проект собирается с помощью Vite-сборщика.
1)Скопировать url проекта: Code => Copy url to clipboard;
2)Открыть Terminal в папке <dirname>, куда будет впоследствии скачан проект, и запустить скачивание проекта => git clone <project-url>;
3)Запустить проект на локальном сервере => npm install(установить npm),
                                           npm run dev (скопировать url сервера и вставить в новое окно браузера)
